# Introduction to Django - LibraryProject

## Project Overview
This project provides hands-on experience with Django, focusing on setting up a development environment, understanding models, and utilizing the admin interface.

## Objectives
* **Set Up Django Development Environment**: Install Django and create a project structure.
* **Implementing Django Models**: Define models and interact with the ORM.
* **Utilizing the Django Admin**: Register models and customize the interface.

## Task 0: Environment Setup
1. **Installation**: Installed Django using pip.
2. **Project Creation**: Created 'LibraryProject' using 'django-admin'.
3. **Development Server**: Launched using 'python manage.py runserver'.

## Core Components
* **manage.py**: Command-line utility for project interaction.
* **settings.py**: Configuration for the Django project.
* **urls.py**: URL declarations and routing.
